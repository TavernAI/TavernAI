## Was TavernAI ist

**TavernAI** erschien 2022, als AI-Chat-Apps noch dabei waren, ihre Form zu finden, und verankerte die Idee: Dialoge und Charaktere bleiben auf dem PC, das AI-Modell ist frei wählbar.

**TavernAI 2** ist ein kompletter Neuaufbau der ersten Version, geprägt von den Erfahrungen der folgenden Jahre. Diese Chats sind nicht mehr auf Gespräche mit einem einzelnen Charakter beschränkt: Sie können zu Szenen mit wechselnder Besetzung, verzweigten Handlungen, visuellen Materialien und interaktiver Logik werden.

**Die Basis bleibt einfach:** Provider verbinden, Charakter auswählen, Nachricht senden. Die Tiefe kommt erst dazu, wenn sie gebraucht wird.

## Hauptfunktionen von TavernAI 2

- **Portable App**: herunterladen und starten, kein Installer nötig. Größe: 300 MB.

- **Branching swipes**: ein Swipe auf einer beliebigen Nachricht erstellt einen weiteren Branch im selben Chat. Jede alternative Antwort kann als eigene Linie weiterlaufen.

- **Multi-Generation**: ein Chat kann mehrere AI-Generationen gleichzeitig ausführen.

- **Dynamische Chat-Teilnehmer**: füge beliebig viele Karten zum Chat hinzu, ändere ihre Reihenfolge, entferne oder ersetze sie, wenn sich die Szene ändert. Der Chat-Verlauf funktioniert weiter, auch wenn sich die Besetzung ändert.

- **Einheitlicher Prompt Manager**: ein System baut das, was das Modell sieht. World info, lorebooks, regex-style replacements und Prompts liegen am selben Ort.

- **Neues Kartenformat**: jede Karte hat ihren eigenen Prompt Manager. Ein Charakter ist nicht mehr auf ein großes Textfeld beschränkt.

- **Anhänge in Prompts**: Prompt Manager kann Dateien und Bilder an jeden Prompt anhängen. Da Karten ihren eigenen Prompt Manager haben, kann eine Karte Charakterbilder enthalten, und die AI kann den Charakter direkt über die Karte sehen.

- **Theme-Editor**: ändere das Aussehen jedes Teils von TavernAI. Erstelle ein Theme für die gesamte Oberfläche, nicht nur eine neue Akzentfarbe.

- **Pre-gen und post-gen Macros**: automatisiere Änderungen vor der AI-Generation und nach der Antwort. Nutze sie, um Prompts, Nachrichteninhalte oder den Szenenzustand im richtigen Moment zu ändern.

- **Prompt Manager Scripts**: machen aus TavernAI 2 eine AI game engine. Füge eigene Panels hinzu, erstelle Mechaniken und mache den Chat zu einer spielbaren AI-Szene.
