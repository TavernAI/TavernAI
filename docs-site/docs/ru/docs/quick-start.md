Быстрый старт ведет от открытия TavernAI 2 до первого ответа AI.

Используйте его после [установки](/ru/docs/installation/). Мы запустим приложение, подключим провайдера, добавим карточку и отправим сообщение.

## 1. Первое сообщение

### 1.1 Запускаем TavernAI

Откройте TavernAI из папки, которую вы распаковали.

![TavernAI launch placeholder](/img/docs/1.png)

### 1.2 Подключаем провайдера

Откройте настройки провайдера и выберите ИИ модель, которую хотите использовать.

![Provider setup placeholder](/img/docs/2.png)


### 1.3 Создаем чат

Нажмите кнопку **Chat** на персонаже, чтобы начать новый чат.

![Create chat placeholder](/img/docs/3.png)

### 1.4 Напишем сообщение

Теперь напишем первое сообщение.

![Add card placeholder](/img/docs/4.png)

### 1.5 Получаем ответ

Теперь получаем первый ответ AI.

![First message placeholder](/img/docs/5.png)

## 2. Добавление карточек

В этот момент мы еще не говорим от имени какой-либо character card. Добавим карточку для себя, чтобы у чата появился участник с нашей стороны.

### 2.1 Открываем панель карточек

Открываем панель добавления карточек в чат.

![First message placeholder](/img/docs/6.png)

### 2.2 Добавляем персонажа

Выбераем карточку, которую хотите использовать для себя.

![First message placeholder](/img/docs/7.png)

### 2.3 Настраиваем карточку

Карточка теперь находится в чате.

![First message placeholder](/img/docs/8.png)

Сейчас AI настроен так, чтобы говорить за обе карточки. Назначим Scout нашей карточкой, а Marika оставим карточкой со стороны AI.

Так Scout выглядит, когда AI говорит за эту карточку:

![First message placeholder](/img/docs/9.png)

А так Scout выглядит после того, как стал нашей карточкой:

![First message placeholder](/img/docs/10.png)

Значения controls карточки:

`AI` — карточки с включенным флагом участвуют в AI-generated messages.

`U` — карточки с включенным флагом участвуют в ответах юзера.

`Ctx` — когда включено, Prompt Manager карточки добавляется в generation context.

`Role selector` — управляет Chat Completion role для сообщений от этой карточки, чтобы модель понимала, считать их сообщениями со стороны пользователя или со стороны AI.

### 2.4 Отправим сообщение от имени карточки

Теперь отправим сообщение как Scout. Marika реагирует иначе, потому что у чата теперь есть персонаж с нашей стороны.

![First message placeholder](/img/docs/11.png)

## 3. Поменяем роли

Теперь создадим новый чат и поменяем настройки карточек местами.

![First message placeholder](/img/docs/12.png)

На этот раз мы играем за Marika, владелицу ночной пекарни, и принимаем Scout как гостя.

![First message placeholder](/img/docs/13.png)
